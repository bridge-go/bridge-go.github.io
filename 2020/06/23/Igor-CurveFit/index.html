Igor CPC标定数据处理程序

2020-06-23
字数统计: 310   |   阅读时长≈ 1 分钟

基本格式：CurveFit [flags] fitType, [kwCWave=coefWaveName,] waveName [flag parameters]

属性：操作符

对象：内置拟合函数

特点：可以不给初值而进行拟合

auto-guess的参数wave为W_coef, 用可选参数kwCWave可以指定初值wave

主要内容：

  • [flags]
  • fitType,
  • waveName
  • [flag parameters]

flags必须在fitType之前，flag parameters必须在waveName之后。

[flags]

/B=pointsPerCycle

拟合函数为sin时可用，pointsPerCycle is the estimated number of data points per sine
wave cycle

/C

约束矩阵

/G

用变量K0, K1 … Kn作为starting guesses进行拟合. 如果用kwCWave关键字指定了初值wave，则starting guesses会从 指定的wave读取

/H="hhh…"

逐个声明拟合参数是否固定。1固定，0变化
h is 1 for coefficients to hold, 0 for coefficients vary.
For example, /H=”100” holds K0 constant, varies K1 and K2.

/K={constants}

设置拟合方程中常量的值，内置函数会自动设置常量，通过此参数可以手动设置。

常数是常数值的list，比如： /K={1,2,3}. 该list的长度必须和拟合方程的常数的数量一致

/N[=dontUpdate]

If dontUpdate = 1, suppresses updates during the fit. This can make the curve fit go
much faster; all graphs, tables, etc. will be updated when the fit finishes. /N is the
same as /N=1.
The default is /N=1. Prior to Igor Pro 7.00 it was /N=0. To update every iteration, use
/N=0.
